# vfox-step-ca

[![CI](https://github.com/jbox-web/vfox-step-ca/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-step-ca/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [step-ca](https://github.com/smallstep/certificates)

## Install

```
mise plugin install step-ca https://github.com/jbox-web/vfox-step-ca.git
```
